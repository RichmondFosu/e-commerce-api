# Products API Project

Django project for e-commerce API.

## Setup
- Django + DRF
- Apps: accounts, categories, products, demo
- Authentication: Token-based
- Database: SQLite

## Configuration
- Settings in `settings.py`
- URLs in `urls.py`
- Static/media files served

See main [README.md](../README.md) and [API.md](../API.md) for usage.
